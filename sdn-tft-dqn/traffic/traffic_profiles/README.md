# Traffic Profiles

This directory contains traffic generation profiles for different scenarios.

## Usage

Traffic profiles can be used to generate different types of network traffic patterns:

- **High bandwidth**: iperf3 with high data rates
- **Medium bandwidth**: iperf3 with moderate data rates
- **HTTP traffic**: Web server requests
- **DNS traffic**: DNS queries
- **ICMP traffic**: Ping requests

## Example Profiles

See `../generate_traffic.sh` for example traffic generation commands.

## Custom Profiles

Create your own traffic profiles by modifying the traffic generation script or creating new profile files.
